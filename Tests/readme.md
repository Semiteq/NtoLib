# Tests

Проект автоматизированных тестов для библиотеки **NtoLib**.

Тесты запускаются через `dotnet test`. Смотри подробнее [документацию](../Build/readme.md) на сборку.

## Назначение

- Проверка корректности ключевых сценариев работы библиотеки.
- Раннее выявление регрессий при изменении кода.
- Поддержка сценариев локальной разработки и CI/CD.

## Структура

- Каталог `Tests` содержит тестовый проект и вспомогательные данные.
- Тесты сгруппированы по функциональным областям, отражающим основные подсистемы библиотеки.

## Запуск

Из корня репозитория:

```bash
dotnet test NtoLib.sln
```

Либо из каталога `Tests`:

```bash
dotnet test
```

Либо из IDE. Rider может потребовать предварительно собрать проект `Tests` чтобы тесты отобразились в окне запуска.

Рекомендуется запускать тесты перед любыми изменениями публичного API или логики библиотеки.
